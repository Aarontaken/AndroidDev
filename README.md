# ActivityFragment
Use the observer pattern to solve the activity and fragment Communications

## 概述

通常来说，解决activity和fragment通信的问题。我们都是采用如下几种方式，
handler，广播，EvnetBus，接口等。本例通过观察者模式解决activity和fragment的通信问题,借鉴了RxJava的思想
